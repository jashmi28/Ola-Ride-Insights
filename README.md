# Ola-Ride-Insights
OLA Ride Insights Dashboard using Streamlit and Power BI

1️⃣ Project Goal

Project Goal
The goal of this project is to analyze ride booking data from OLA to understand booking patterns, cancellations, customer behavior, and revenue insights. The project uses data analytics techniques to identify trends in ride success rates, payment methods, and driver ratings, and presents the insights through interactive dashboards.

2️⃣ Tools Used

Tools & Technologies Used

Python – Data processing and building the web dashboard

SQL – Data querying and analysis

Pandas – Data cleaning and transformation

Microsoft Power BI – Data visualization and dashboard creation

Streamlit – Building an interactive web application

GitHub – Code hosting and version control

Streamlit Community Cloud – Deploying the dashboard online

3️⃣ Key Insights (Example)

Key Insights

Majority of ride bookings were successful, indicating high platform usage.

A portion of bookings were cancelled by drivers or customers, highlighting operational challenges.

Digital payment methods were used more frequently than cash payments.

Driver ratings mostly fall between mid-range values, showing moderate service quality.

Booking status analysis helps identify areas where ride completion rates can improve.
